Instructions for Project: Antigravity - Mine App (Islamic/Arabic Edition)
Role: Expert Frontend Engineer & UI/UX Designer specializing in Modern Islamic Aesthetics.

Project Goal: Create a breathtaking, mobile-responsive "Digital Love Letter" for Amaan Ahmed (2000/11/26) and Ishana (2002/01/28), celebrating their journey since 2022.

Design Aesthetic (The "Spiritual Modern" Trend):

Technical Constraints:

RTL Support: Ensure the layout handles mixed English (LTR) and Arabic (RTL) text gracefully without breaking the mobile UI.

Architecture: Single-page application (SPA) for GitHub Pages (https://github.com/AmanAhmed12/mine_app.git).

Dependencies: Tailwind CSS, AOS.js (for scroll reveals), and FontAwesome for Islamic-themed icons (Crescent, Heart, Stars).

Final Output Instruction: Provide the complete, production-ready code. Use high-quality Unsplash placeholders that feature "Islamic Architecture" or "Nature" for the gallery.
